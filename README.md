> **"Find help faster. Become help that matters."**

**Helplytics AI** (Branded as **HelpHub AI**) is a premium, community-powered support network designed for students, mentors, creators, and builders. This project was developed as part of the **SMIT Grand Coding Night (April 2026)** for **Saylani Batch 18**, focusing on a high-fidelity, multi-page UI/UX experience.

---

## 🚀 Live Product Feel
This project is more than just a collection of forms; it's a simulated ecosystem. It features a polished multi-page experience inspired by modern product platforms, with AI summaries, trust scores, contribution signals, and notification systems built entirely with native web technologies.

---

## ✨ Key Features (UI-Driven)

### 🤖 Smart AI Center
- **Request Intelligence**: Simulated auto-categorization and urgency detection for help requests.
- **Rewrite Suggestions**: AI-driven guidance to turn rough problems into clear, solvable requests.
- **Trend Detection**: Surfaces rising community demands and skill gaps.

### 📊 Community Trust Graph
- **Dynamic Trust Scores**: Track reputation based on solved requests and community activity.
- **Helper Rankings**: Visualizing top contributors and mentors via a leaderboard.
- **Contribution Badges**: Recognition for various roles like "Design Ally", "Fast Responder", and "Top Mentor".

### 📡 Live Community Pulse
- **Explore Feed**: A real-time view of community problems currently in motion.
- **Live Signals**: Interactive notification system for matches, reputation boosts, and insights.
- **Messaging ecosystem**: Direct paths to move from a match to a solution.

### 🛠️ Advanced Admin Panel
- **Moderation Table**: Oversight for all platform requests and their status.
- **Platform Analytics**: High-level metrics for user growth, request volume, and solution rates.

---

## 🛠️ Technology Stack

- **Core**: HTML5, Semantic Markup
- **Styling**: Native CSS3 (Custom Design System, Glassmorphism elements)
- **Logic**: Vanilla JavaScript (ES6+)
- **Persistence**: LocalStorage (Session management and data simulation)
- **Deployment**: [Vercel](https://helplytics-ai-one.vercel.app/)
